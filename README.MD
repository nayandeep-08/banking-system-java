# Banking Management System (Java)

A console-based banking system built using Java to practice backend fundamentals.

## Features
- User registration & login
- Savings & Current accounts
- Deposit & Withdraw
- Transaction history
- File handling (data persistence)

## Tech Used
- Java
- OOP
- Collections (HashMap, List)
- File Handling (Serialization)

## How to Run
1. Clone the repo
2. Open in IntelliJ/Eclipse
3. Run Main.java
